Below is a list of popular text editors. For our class, we will focus primarily on Visual Studio Code.

  • Microsoft Visual Studio Code - A forked variation of the Atom editor by Microsoft. Free.
  • Adobe Dreamweaver - an Adobe product that's part of Creative Cloud. Known for its WYSIWYG interface, its features can be considered a bit bloated.
  • Adobe Brackets - Open source underpinning for the deprecated Adobe Edge Code (2014). Extendable via extensions.
  • Sublime Text - another simple, popular text editor.

Plugins/Extensions/Packages

The beauty of many text editors today is they allow customized functionality. Here are a few of my favorites:

  • Emmet - an extremely popular plugin that assists in autocompletion of code.
  • Fira Code - a nice code font with programming ligatures.

For VS Code

(install via Code > Preferences > Extensions):

  • Bracket Pair Colorizer - A customizable extension for colorizing matching brackets
  • Beautify - Beautify code in place for VS Code
  • Color Highlight - Highlight web colors in your editor
  • Document This - Automatically generates detailed JSDoc comments in TypeScript and JavaScript files
  • ESLint - Integrates ESLint JavaScript into VS Code
  • Fira Code - Monospaced font with programming ligatures
  • Live Server - Launch a development local Server with live reload feature for static & dynamic pages
  • vscode-icons - Icons for Visual Studio Code

For Brackets

(install via File > Extension Manager... > Available tab):

  • Brackets Icons - Creates icons for files in the left menu
  • Brackets SASS - supports SASS
  • Color Highlighter - highlights color values with colors in CSS code
  • Documents Toolbar - adds tabs to interface
  • Minimap - displays a preview panel of the entire document
