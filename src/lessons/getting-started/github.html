GitHub is a service that provides software development hosting and version control using Git. It enables collaboration within a team by sharing code and tracking individual changes.

Getting Started
  1. Register an account with https://www.github.com/
  2. Download the GitHub desktop app: https://desktop.github.com/
  3. Register your account (I recommend you remain logged into github.com on your browser, as it makes it easier to authenticate)
  4. Click “Create a New Respository on your Hard Drive…"
  5. Name it “exercise1” and click "Create Repository"
  6. Create an index.html page within the exercise1 folder. Save your file, then click "Commit to main"
  7. Click "Publish repository". This will publish the project to GitHub's website.
