Software

For our class, we will use Microsoft Visual Studio Code (VS Code) as our text editor.

You'll also need the Live Server extension, which automatically refreshes the browser whenever we save our files in VS Code.

Browser Inspectors

Familiarize yourself with browser inspectors, as these tools will prove essential in writing and troubleshooting your code.

  • Firefox Page Inspector
  • Chrome Dev Tools
  • Safari Developer Tools

Github
